## Info
Rawarr is a cross-platform, free and open source music player for Navidrome or other [Subsonic](http://www.subsonic.org) servers. Powered by React Native.

## Features
- Offline support
- CarPlay support
- AI queue creation
- Lidarr integration
- Dark mode support
